QGenda Calendar Helper - Privacy Policy

Last updated: December 2024

This privacy policy describes how QGenda Calendar Helper ("we", "our", or "us") handles information when you use our Chrome extension.

Information We Access

  • QGenda iCal URLs that you explicitly provide through the extension interface
  • Calendar data from QGenda (only when you configure a valid iCal URL)
  • Website URLs to detect compatible timesheet and scheduling applications

Information We Do NOT Collect

  • Personal information beyond what's contained in your QGenda calendar data
  • Browsing history outside of extension functionality
  • Analytics, usage tracking, or behavioral data
  • Any data from websites unrelated to timesheet/scheduling applications
  • Financial information or creditworthiness data

Data Storage and Security

  • Local Storage Only: All calendar data is stored locally in your browser using Chrome's secure extension storage API
  • No Cloud Storage: No data is transmitted to our servers or cloud services
  • No Third-Party Sharing: Data is never shared with external services or companies
  • Secure Communication: All data exchange with web applications uses Chrome's secure postMessage API

Data Usage

We use the accessed data solely for the extension's single purpose:

  • Fetching calendar data from your configured QGenda iCal URL
  • Storing calendar data locally for quick access
  • Providing calendar data to compatible timesheet/scheduling web applications
  • Automatically syncing calendar data for convenience

Data Sharing

  • Same-Origin Only: Calendar data is only shared with web applications on the same domain you're visiting
  • User Control: Data sharing only occurs when you're using compatible timesheet applications
  • No Third Parties: We never share data with third-party services, advertisers, or other companies
  • No Analytics: We don't use analytics services that would access your data

Your Rights and Control

  • Configuration Control: You decide which QGenda URL the extension accesses
  • Data Deletion: You can clear all stored data using the "Clear Stored Data" button in the extension
  • Complete Removal: Uninstalling the extension removes all associated data
  • URL Changes: You can change or remove your QGenda URL at any time

Permissions Explanation

Our extension requests minimal permissions for its functionality:

  • "storage": Store your QGenda URL settings and calendar data locally
  • "activeTab": Communicate with timesheet applications on the current tab
  • "tabs": Detect timesheet applications for automatic sync
  • "https://app.qgenda.com/*": Access only QGenda iCal URLs you provide

Data Retention

  • Calendar data is retained locally until you clear it or uninstall the extension
  • No data is retained on external servers
  • You have complete control over data retention through the extension interface

Security Measures

  • Direct HTTPS connections to QGenda (no third-party proxies)
  • Chrome's secure extension storage API
  • Same-origin policy enforcement for data sharing
  • Minimal permission requests
  • No external network connections except to your configured QGenda URL

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last updated" date. We encourage you to review this policy periodically.

Compliance

This extension and privacy policy are designed to comply with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
  • Healthcare data security best practices
